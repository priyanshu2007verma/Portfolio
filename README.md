# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/priyanshu2007verma/Portfolio.git
```

Go to the project folder:

```bash
cd Priyanshu-Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
